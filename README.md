# playground

Experiments, workshops & fun projects — where ideas get tested.

An open sandbox for prototypes, learning projects, and workshop materials that don't
fit neatly into professional or research work.

## What lives here

**ai-lab**  
Explorations with Claude Code and AI-assisted workflows. Started as a Bauhaus workshop
project and still actively evolving.

**supervision**  
Pedestrian detection and counting using computer vision (Roboflow). Built as a proof
of concept in 2023.

**Workshop materials**  
Teaching resources and exercise code from seminars on computational design, parametric
urban analysis, and open data.

**Prototypes**  
Early-stage ideas, quick experiments, things that may or may not go anywhere.

## The idea

Not everything needs to be polished. This is the space for trying things — tools
half-built, ideas half-baked, learning in public.
